# Web Development Mini-Tasks Challenge

A premium, interactive web application designed to challenge students with 30 mini-tasks covering HTML, CSS, and basic JavaScript.

## Features
- **30 Unique Tasks:** Ranging from basic HTML tags to CSS Grid and JS DOM manipulation.
- **Glassmorphic UI:** A modern, sleek design with animated background blobs and soft transitions.
- **Interactive Timer:** 15-minute countdown with a 3-minute overtime period.
- **Personalized Experience:** Users enter their name before starting a task.
- **Responsive Design:** Works seamlessly on mobile, tablet, and desktop.

## How to Use
1. Open `index.html` in your browser.
2. Select any number from the grid.
3. Enter your name in the modal and click **Start**.
4. Read your task and complete it within the 15+3 minute window!

## Topics Covered
- **Week 1:** HTML Structure, Tags, Semantic Elements, Accessibility.
- **Week 2:** CSS Basics, Box Model, Flexbox.
- **Week 3:** Advanced CSS, Positioning, Z-index, CSS Grid, Responsive Design.
- **Week 4:** CSS Animations, Keyframes, Transforms, JS Introduction.
